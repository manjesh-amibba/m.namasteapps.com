Version 5.4 - Sticky Sidebars - Update Guide

Version 5.4 brings new sidebar designs to Sticky, and with these a few minor changes that must be applied

Via SCSS

If you're updating your CSS using SCSS the following files have changed.

/scss/sticky/_menu.scss

Via CSS

Open style.css and, on line 883 add the code you find in the update style css. The new lines added are lines 883 to 941

New JavaScript

We also updated the menu-hider / backdrop to improve performance and fix a minor bug that was affecting a few customers in certain situations.

In your project, open scripts/custom.js and copy lines 27 to to 30 from the updated style.css

Having issues updating?

If you're facing any issues updating to the new version of Sticky, please send us a message via our contact form (link). We'll gladly check your issue and help out in the shortest possible time ( usually less than 2 hours unless we're sleeping )
