Version 4.6 - Sticky Admin - Update Guide

Version 4.6 brings a few optimisations to the code of Sticky. Here is an exact step by step update procedure.

Via SCSS

If you're updating your CSS using SCSS the following files have changed.

/scss/sticky/_footer-bar
/scss/components/_carousels
/scss/components/_cards
/scss/components/_search

Via CSS

If you're updating your CSS in the classic way, then, download the main package from TF, and add/replace the following lines from style.css in your style.css.

Add lines 2345 to 2347 to your corresponding code in style.css
Replace lines 4179 to 4228 to your corresponding code in style.css
Add line 3866 in your corresponding code in style.css

New JavaScript

In order to get the best performance from the admin pages using charts JS, these are called only on pages where the class='graph' is located. This is detected via JS. The JS functions for the charts can be found inside the /plugins/charts folder.

Step 1 In your project, find and delete the plugins/charts/ folder and replace it with the one from Sticky 4.6
Step 2 In custom.js in Sticky 4.6 locate this code - https://jmp.sh/yhVj76C on line 1311. In your custom.js locate the same code and replace it with lines from 1311 to 1355 from Sticky 4.6. You will be replacing the entire array.
Step 3 Also, in Sticky 4.6, we are now tracking iOS15 in order to easily adjust future updates for the new design. This is done via the function in custom.js from lines 303 to 313. Copy these lines to your project as well.
